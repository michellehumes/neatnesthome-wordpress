# Image Requirements for Neat Nest Home

## Category Images (for homepage grid)

Upload these to `/assets/images/categories/`:

| Filename | Dimensions | Description |
|----------|------------|-------------|
| `category-kitchen.jpg` | 600x400px | Organized pantry or kitchen shelves |
| `category-closet.jpg` | 600x400px | Tidy closet with organized clothing |
| `category-bathroom.jpg` | 600x400px | Clean, organized bathroom storage |
| `category-small-spaces.jpg` | 600x400px | Apartment or small room organization |
| `category-kids.jpg` | 600x400px | Organized kids room or playroom |
| `category-entryway.jpg` | 600x400px | Organized entryway with hooks/baskets |
| `category-laundry.jpg` | 600x400px | Organized laundry room |
| `category-garage.jpg` | 600x400px | Organized garage or seasonal storage |

**Style notes:**
- Bright, natural lighting
- Clean, minimal aesthetic
- Real spaces (not overly staged)
- Neutral color palette that matches brand

## Product Images

For product reviews and roundups:

| Type | Dimensions | Notes |
|------|------------|-------|
| Featured product | 800x800px | Square, white or light background |
| In-content product | 600x400px | Can be lifestyle or product-only |
| Comparison table | 300x300px | Small, clear product shots |

**Sources for product images:**
- Amazon product images (with attribution)
- Manufacturer press images
- Your own photography
- Canva product mockups

## Image Optimization Checklist

Before uploading any image:

- [ ] Compress using TinyPNG or ShortPixel
- [ ] Use descriptive filename (e.g., `best-pantry-organizer-vtopmart-bins.jpg`)
- [ ] Target file size: under 100KB for most images
- [ ] Use WebP format when possible (with JPG fallback)
- [ ] Add alt text with keywords in WordPress

## Recommended Tools

- **Canva**: Create graphics, Pinterest pins, and simple product collages
- **TinyPNG/TinyJPG**: Compress images before upload
- **Remove.bg**: Remove backgrounds from product images
- **Unsplash/Pexels**: Free stock photos (check license)
